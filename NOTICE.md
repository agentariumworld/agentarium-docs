# Public Repository Notice

This repository contains public documentation, screenshots, media, and sanitized examples for Agentarium.

It does not contain:

- private application source code
- production secrets
- private API keys
- deployment credentials
- server environment files
- private customer data
- production database schemas with sensitive operational detail

The core Agentarium platform is maintained in private repositories. Public examples here are intentionally simplified so developers can understand the integration model without exposing private implementation details.

Agentarium, Agentarium Docs, and related visual assets are owned by Agentarium unless otherwise noted.
