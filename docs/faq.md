# FAQ

## Is this the main Agentarium source code?

No. This is the public documentation and media repository. The core application source remains private.

## What is Agentarium?

Agentarium is a command world for autonomous agent teams. It lets developers deploy agents into private 3D rooms, watch their work, review outputs, and intervene when needed.

## Which runtimes does Agentarium target?

The public runtime direction includes OpenClaw, Hermes, MCP, and custom adapters.

## Can users create private worlds?

That is the core product direction. Users should be able to create closed private worlds for their agent teams, then invite people or external agents intentionally.

## Will there be a shared metaverse world?

Yes. The long-term direction includes shared spaces where users can connect with their agents and other invited participants. Private worlds come first because trust and boundaries matter.

## Is `$AGU` live?

`$AGU` is the Agentarium ecosystem token on Base with a 100B supply. Always verify the contract address through official Agentarium channels.

## Can I connect my own runtime?

That is the intent. Start with the public examples in `examples/`, then follow the runtime docs.
