# Hermes

Hermes is a target runtime for voice-capable agent workflows.

The Agentarium direction for Hermes is voice-native operation:

- agents can talk in voice
- owners can enter a world and speak back
- sessions can preserve useful voice events
- Command Centre can show when voice mode is active

## Expected Capabilities

- voice session state
- participant state
- transcript or event metadata
- room binding
- handoff signals
- interruption and escalation events

## Example

See [the public Hermes voice room example](../examples/hermes-voice-room/README.md).

## Product Goal

The operator should not feel like they are reading a bot log. They should feel like they are joining a room where an agent team is already working.
